Introduction

Timeline is a small web component that enables rendering timelines. The layout is customizable. Examples are shown below.

To use this component it is sufficient to include the following code in your HTML file: <script type="module" src="timeline.js"></script>

The styling is done using the following CSS.

Please, keep in mind that you are primarily styling SVG elements. Only the ld-timeline element is an HTMLElement. 
<style>
    /* Demonstrates optional styling! */
    ld-timeline.html-timeline {
        box-sizing: border-box;
        padding: 20px;
        background-color: rgba(0, 0, 0, 0.25);
        box-shadow: 4px 4px 6px rgba(0, 0, 0, 0.5);
        border-radius: 0.5em;

        &::part(pane) {
            font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;
            opacity: 0.75;
            padding: 10px;
            fill: white;
        }

        &::part(date) {
            font-size: 20px;
            font-weight: lighter;
        }

        &::part(description) {
            font-size: 24px;
        }

        &::part(timeline-main) {
            stroke: whitesmoke;
        }

        &::part(timeline-suffix) {
            stroke: whitesmoke;
        }

        &::part(timeline-prefix) {
            stroke: whitesmoke;
        }

        &::part(date-marker) {
            stroke: whitesmoke;
        }
    }
</style>

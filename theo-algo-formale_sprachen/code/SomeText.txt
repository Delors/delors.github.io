# Mein Persönliches Markup
Mein persönliches Markup dient der schnellen Auszeichnung von Texten.

## Überschriften
Überschriften auf Ebene 1 werden durch eine # am Anfang der Zeile ausgezeichnet. Überschriften auf Ebene zwei durch zwei # am Anfang der Zeile.

Beispiel:
:: # Überschrift auf Ebene >1<
:: 
::
:: ## Überschrift auf Ebene >2<

## Escape Character
Mein Escape Zeichen ist das Backslash (\\). Das Zeichen nach einem Backslash wird immer direkt ausgegeben. Um ein Backslash auszugeben nutze zwei Backslashes: \\\\.

## Fett
Worte oder Satzteile (innerhalb einer Zeile), die *direkt* in zwei "*" eingeschlossen werden und nicht mittels \\ escaped sind, werden in Fett geschrieben. 

Beispielsweise: \*fett\* => *fett*, aber auch \*F\* => *F*.

Allerdings würde: 2 * 3 * 4 nichts fett auszeichnen, da die * nichts direkt einschließen. Zwei Sterne hintereinander: ** werden ungeändert ausgegeben.

## Monspaced
Eine Zeile Text wird Monospaced ausgegeben durch ein "::" am Anfang der Zeile. Ein einzelnes Whitespace nach dem "::" wird ggf. ignoriert. Ein "::" ohne weiteren Text erzeugt eine Leerzeile.

## Kommentare
Ein Kommentar (* Frage: ist die Wahl dieser Sonderzeichen für Kommentare gut? *) startet immer mit \(\* und endet mit \*\). Ein Kommentar kann überall vorkommen erstreckt sich aber nicht über mehrere Zeilen. Verschachtelung wird nicht unterstützt. D. h. das erste \*\), dass auf ein \(\* folgt, beendet einen Kommentar, auch wenn es mehrere \(\* gab.

(* TEST (*(* (* Kommentar - Ein <-- einfacher --> Kommentar. *)

# Leere Zeilen
Leere Zeilen erzeugen in der Ausgaben einen Zeilenumbruch.
